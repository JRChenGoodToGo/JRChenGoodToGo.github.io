音樂活動為主軸，提供給音樂人分享演出活動及參加活動的平台。 <br>
主要呈現 <br>
-使用者進入網頁時會優先看到熱門的前幾項活動及點擊次數，增添演出者為了搏版面而宣傳活動的趣味。 <br>
-篩選活動功能，呈現使用者所選擇地區或月分所舉辦的活動以快速瀏覽對使用者而言更有用的活動。 <br>
-能收藏使用者喜愛的活動的功能，讓使用者可以隨時看到喜愛活動的最新更新情況。 <br>
-清晰的版面規劃，讓會員可以快速的根據規劃流程新增或編輯活動資訊。 <br>
